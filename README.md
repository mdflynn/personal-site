# Mike Flynn Codes

###### An interactive, online resume and portfolio

---

## Overview

This project was a personal goal of mine. I wanted to have my own website to be able to share my career accomplishments in an interactive format mikeflynncodes.com

## Goals

- Demonstrate React and Sass skills
- Tell career story through an online format
- Practice incorporating new packages through npm

## Wins

- Deploying GitHub page to personal domain on Google Domains
- Successful integration of npm packages
- Robust testing with CI
- Lots of fun playing with animations

## Challenges

- Integrating GitHub pages with a personal domain. Remember to update package.json!
- Creating an internal contact form that generates an email that is automatically sent on submission
- Ensuring SASS was not conflicting across the many animations in the app
